# Summary

Generate summaries of activities, conversations, and progress.

## When to use

Use this skill when the user asks to:
- Generate a daily summary
- Create a weekly report
- Summarize recent activities
- Review progress on goals

## Usage

Generate daily summary:
```
Review recent messages and activities from the conversation history.
Check GOALS.md for progress on active goals.
Summarize key accomplishments and pending items.
```

Create weekly report:
```
Review the past week's activities.
Check goal progress and deadlines.
Summarize achievements and blockers.
```

## Best Practices

- Focus on key accomplishments and decisions
- Highlight pending items and deadlines
- Reference specific goals from GOALS.md
- Keep summaries concise and actionable
- Note any blockers or risks
